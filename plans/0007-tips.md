# Phase 6 — Tips ❌

## Goal
Static IELTS improvement tips, filterable by skill and band target.
Read-only for users. Write via admin endpoints (Phase 7).

## Prerequisite
Phase 1 (schema) complete.

## API Endpoints

### List tips
```
GET /api/v1/tips
Query params: skill (optional), band_range (optional)
Response: { tips: [{ id, skill, title, content, band_range }] }
```

## Domain Types (`internal/domain/tip.go`)
```go
type Tip struct {
    ID        string
    Skill     string
    Title     string
    Content   string
    BandRange string    // e.g. "5-6", "7+"
    CreatedAt time.Time
}
```

## Repository Interface
```go
type TipRepository interface {
    ListTips(ctx context.Context, skill, bandRange string) ([]Tip, error)
}
```
